Typography System — CD / CI

Georgia meets DM Sans — authority meets clarity.

Georgia carries the brand's editorial weight: headings, display text, and the logo all use Georgia with carefully tuned letter-spacing corrections. DM Sans handles everything that needs to be read at length — body copy, labels, UI, captions. The two typefaces are distinct but complementary: one speaks in authority, the other in clarity.

Why Georgia works for Rautaki:
Designed by Matthew Carter for screen rendering, Georgia has excellent legibility at all sizes, generous letterforms, and strong contrast between thick and thin strokes — all qualities that reinforce the brand's character. It is a system font, meaning zero loading time on any device.

The kerning rule:
Georgia is calibrated for body text. At display sizes (20px+), optical letter-spacing must be corrected downward. Each heading level in this system has its kerning pre-set as a CSS variable.
